Polyline Drawing Tool

This plugin allows users to draw polylines (connected line segments) on the chart by clicking to add points. The tool provides buttons directly on the chart for better user interaction.

  • Polyline Button: Toggle drawing mode (top-right of chart)
  • Color Picker: Change the line and point color
  • Clear Button: Remove all points
  • Line Style Dropdown: Choose solid or dashed line

To draw a polyline:

  1. Click the polyline button in the top-right of the chart
  2. Click anywhere on the chart to add points
  3. The line will preview as you move your mouse
  4. Click the polyline button again to stop drawing
